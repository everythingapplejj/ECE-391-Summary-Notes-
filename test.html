How to View Your RISC-V Quiz

A step-by-step guide to opening and testing your HTML file

Steps to View Your HTML File

1Save Your HTML File

Make sure your file is saved with a .html extension (e.g., riscv-quiz.html).

2Locate the File

Find where you saved the HTML file on your computer.

3Open with Browser

Right-click the file and choose "Open with" then select your web browser (Chrome, Firefox, Edge, etc.).

Tip: You can also open your browser and drag the HTML file directly into the browser window.

Troubleshooting Common Issues

File Won't Open

Make sure your file has the .html extension. If it has a .txt extension, rename it.

Page Doesn't Display Correctly

Check the browser console for errors (F12 key). There might be syntax errors in your HTML, CSS, or JavaScript.

Code Doesn't Run

Ensure JavaScript is enabled in your browser. Some browsers block JavaScript from local files for security.

Pro Tip: For better development experience, use a code editor like VS Code with the Live Server extension, which automatically reloads your page when you make changes.
